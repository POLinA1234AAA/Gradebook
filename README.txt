The E-Journal Management System is a software application developed using .NET that streamlines the process of tracking student academic performance, managing user roles, and facilitating efficient communication within educational institutions.

## Introduction

In today's educational environment, educators and administrative staff often spend a significant amount of time on manual record-keeping tasks related to student performance and course management. The E-Journal Management System aims to automate and simplify these tasks, allowing educational institutions to:

- Efficiently track student performance and progress.
- Manage user roles, including administrators, teachers, and students.
- Enhance transparency and communication within the educational process.

## Features

### User Management

- **User Roles**: The system supports three user roles: System Administrator, Teacher, and Student.
- **Registration**: Users can register based on their roles, providing necessary details.
- **User Authentication**: Secure login and authentication mechanisms for all users.

### System Administrator Features

- **User Management**: System administrators can add, delete, or edit user accounts, including their registration details.
- **System Log**: The admin can view a log of user login/logout activities and data entry operations.

### Teacher Features

- **Course Management**: Teachers can manage course-related information, including disciplines, groups, and students.
- **Grade Entry**: Ability to input and update student grades and academic records.
- **Reports**: Generate reports such as student performance summaries and class rosters.

### Student Features

- **View Grades**: Students can access their academic records and view grades for each course.
- **Course Information**: Access course information, including schedules and assigned teachers.
- **Academic Progress**: Track academic progress and view course history.

## System Requirements

- **Operating System**: Windows (compatible with .NET Framework).
- **Development Tools**: Visual Studio or a compatible .NET IDE.
- **Database**: SQL Server or another compatible database system.

## Usage

1. Launch the application and register as a user based on your role.

2. Access the features based on your user role.

3. Use the system to manage student records, grades, and course information.
